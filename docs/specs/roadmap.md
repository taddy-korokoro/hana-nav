### スケジュール遅延時の削減優先度

```
削減可（高→低）：
1. アフィリエイトリンク埋め込み
2. リワード広告連携（ロジックは仕込むが UI なしでも OK）
3. 簡易レビュー機能
4. 地図 UI（リスト表示のみで代替）
5. Google OAuth（Email ログインのみで代替）

削減禁止（コアバリュー）：
- スポット検索（エリア/花種類/見頃）
- スポット詳細表示
- ブックマーク機能
- AI 花判定
- 旅のしおり画像生成 + SNS シェア
- レート制限
```
